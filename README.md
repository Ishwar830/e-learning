# E-Learning Platform

## Description

Simple e-learning SPA using React Router
Live Page: <https://e-learning-2rmn.onrender.com>

## Features

* Simple user authentication
* Search and filter different courses
* Users can enroll in a course
* Users can view courses they have enrolled into

## Frameworks and Libraries used

* Vite
* React
* React-Router
* TailwindCSS
* Lucide Icons

## Learnings

* React Router
* Handling Routes
* Routes with dynamic parameters
* Custom Hooks
* Debouncing Search Input for better performance
